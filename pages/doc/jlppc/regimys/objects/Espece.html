Classe jlppc.regimys.objects.Espece

La classe qui définit une espèce de Pokémon.

public class Espece implements Serializable

Variables :
  • protected final String surnom : La nom de l'espèce. Cette variable s'appelle surnom et pas nom par erreur lors de sa creation
  • protected int numeropokedex : Le numéro Pokédex de L'espece
  • protected Espece evolution : L'espèce en lequel le pokémon évolue
  • protected int niveauEvolution : Le niveau auquel cette espèce evolue.
  • public Evolution evolType : Le moyen d'évolution de l'epèce.
  • protected final Type type1 : Le type principal de l'espece
  • protected final Type type1 : Le type secondaire de l'espèce. Si cette espèce n'a qu'un seul type, il faut mettre aucun dans cette variable (ET SURTOUT PAS NULL)
  • protected HashArray[] atksByLevels : Les attaques apprises par niveau. Des tags sont ajoutés : "niveau" pour l'ID 0 et "attaque" pour l'ID 1.
  • protected String entreePokedex : La description du pokémon dans le pokedex
  • protected float taille : La taille du pokémon
  • protected float poids : Le poids du pokémon
  • protected CT[] ctCompatibles : Les CT compatibles avec l'espece
  • protected final int baseStat : Remplacer Stat par Atk, Def, AtkSpe, DefSpe, Vit, PV. Les stats de base de l'espèce.
  • protected int hashCode : Le hashCode de l'espece
  • protected CourbeExp courbe : La courbe d'experience que le pokémon suit pour monter de niveau
  • protected int expMax : L'xp du pokémon au niveau 100
  • protected Stats[] EVgiven : Les EV donnés en fin de combat
  • protected int expGiven : Les exp donnés de base si un pokémon de cette espèce est vaincu
  • protected int tauxDeCapture : Le coefficient de capture du pokémon
