  • public void setStat(String stat, int newStat) : Permet de changer une statistique. A utiliser avec modération.
    Arguments :
    • stat : La statistique a changer. Elle peut prendre les valeurs "atk", "def", "atkspe", "defspe", "vit", "pv" selon la statistique a changer.
    • newStat : La nouvelle valeur a affecter a cette statistique

  • public synchronized void setStats(int[] stats, Attaque[] attacks, Espece espece, Type[] types) : Permet de changer toutes les stats avec les arguments. L'esquive et la precision sont remis a 100.
    Arguments :
    • stats : Les statistiques dans cet ordre : atk, def, atkspe, defspe, vit.
    • attacks : Les 4 attaques du pokémon
    • espece : L'espece du pokémon. Le pokemon continuera a porter le nom de son acienne espece si il n'a pas été renommé.
    • types : Les deux types dans l'ordre
  • Encore des setters et des getters...
  • public synchronized void attaqued(int pvPerdus) : Methode appelée lorsque le pokémon est attaqué.
    Arguments :

  • public boolean change*stat*(int power) : Changer *stat* par ATK, DEF, ATKSPE, DEFSPE, VIT, ESQ ou PRE dans le nom donné. Ces methodes permettent d'augmenter ou de diminuer leur stat respective par rapport aux niveaux (voir Pokepedia).
    Arugments :
    • power : Le nombre de niveaux à augmenter. Si ce doit etre diminuer, entrez un nombre negatif.
    Retourne : true si l'augmentation de stat a bien eu lieu, false si elle était trop haute ou trop basse.
  • public boolean setStatus(Status status) : Permet de changer le status.
    Arguments :
    • status : Le status a mettre
    Retourne : true si le changement de status a bien eu lieu, false si il y avait deja un status.
