Classes internes :
  • public abstract static class CalcCourbes implements Serializable : Classe permettant de calculer les courbes d'experience. Annotation : @Static. La classe est protected, donc les methodes aussi (bien que notées public)
  • public enum Status implements Serializable : Enumération des status des pokémon. Liste des enumerations :
    • BRULURE
    • PARALYSIE
    • SOMMEIL
    • GEL
    • POISON
    • AUCUN
Constructeurs :
  • public Pokemon(Espece espece, int level, Attaque[] attaques, Caractere caractere, Talent talent) : Constructeur créant un pokémon sans surnom. Les noms des données correspondent a leurs variables.
  • public Pokemon(String surnom, Espece espece, int level, Attaque[] attaques, Caractere caractere, Talent talent) : Constructeur créant un pokémon avec un surnom. Les noms des données correspondent a leurs variables.
Methodes :
  • public boolean captured(I_Pokeball pokeball) : Verifie si le pokémon est capturé. Paramètres :
    • pokeball : La pokeball utilisée
    Retourne : true si le pokemon est capturé, false sinon.
  • public void setStat(String stat, int newStat) : Permet de changer une statistique. A utiliser avec modération. Paramètres :
    • stat : La statistique a changer. Elle peut prendre les valeurs "atk", "def", "atkspe", "defspe", "vit", "pv" selon la statistique a changer.
    • newStat : La nouvelle valeur a affecter a cette statistique
